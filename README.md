# NextShape

Este projeto consiste no desenvolvimento de um e-commerce voltado para a venda de suplementos, com o objetivo de praticar a construção de interfaces web e a organização de um sistema de loja online.

A aplicação foi desenvolvida utilizando HTML, CSS e JavaScript, com foco na criação de um layout moderno, estruturação de páginas e navegação entre seções, simulando a experiência de um usuário em uma loja virtual.

O projeto também tem como objetivo aplicar conceitos de organização de código no front-end, além de servir como base para futuras implementações mais avançadas.

## Funcionalidades
- Página inicial com apresentação da loja
- Exibição de produtos
- Estrutura de navegação entre seções
- Carrinho de Compras
## Tecnologias utilizadas
- HTML
- CSS
- JavaScript
## Melhorias futuras
- Integração com backend
- Sistema de login e cadastro de usuários funcional
- Persistência de dados (produtos, pedidos, usuários)
- Integração com API e banco de dados
